# DevOps Docker Nginx Project

This project demonstrates a simple DevOps architecture using Docker and Docker Compose.

## Architecture

User → NGINX → Application → MySQL Database

## Technologies

- Docker
- Docker Compose
- NGINX
- Node.js
- MySQL

## Project Structure

devops-docker-nginx-project
│
├── docker-compose.yml
└── README.md

## Run the project

docker-compose up -d

## Access the application

http://localhost:8080
